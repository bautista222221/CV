Descripción del proyecto

Plataforma full stack que permite generar cursos de programación de forma automatizada. Corrige y da feedback inmediato mediante técnicas de IA.

Características principales

  • Generación automática de cursos, unidades, ejercicios y evaluaciones
  • Integración entre frontend y backend mediante APIs RESTful
  • Correccion automatizada de ejercicios y evaluaciones con feedback inteligente
  • Integración con APIs de Mercado Pago y Paypal
  • Modelo de suscripciones para acceso ilimitado al sistema
  • Posibilidad de uso gratuito con limitación en la cantidad de cursos creados/vistos
  • Emisión de certicaciones PDF para cursos completados exitosamente
  • Arquitectura escalable tanto en front-back como con los nodos de correccion
  • despliegue On Demand de nodos de correccion
  • implementacion de Auth0 para la autenticación de usuarios
  • Totalidad del sistema desplegado en cloud

Tecnologías utilizadas

  • Frontend: Vue.js
  • Backend: Spring Boot

Descripción del proyecto

Plataforma web que permite a los usuarios registrarse y cursar materias, realizar evaluaciones, y recibir recomendaciones personalizadas basadas en algoritmos de Inteligencia Artificial.

Características principales

  • Registro y autenticación de usuarios
  • Creación de cursos y evaluaciones
  • IA para sugerencia de contenidos
  • Contabilización de usuarios inscriptos a cada curso
  • Diseño adaptable a dispositivos móviles

Galería de imágenes

Tecnologías utilizadas

  • Frontend: HTML, CSS, JavaScript
  • Backend: PHP
  • Base de datos: PostgreSQL
  • IA: Llama, modelo llama-3-8b-instruct

Descripción del proyecto

Plataforma web para la gestión de una librería. Permite el manejo de usuarios, catálogos de libros y reservas. Implementa autenticación y funcionalidades CRUD para cada entidad.

Características principales

  • Gestión completa de usuarios y libros
  • Sistema de reservas de ejemplares, con persistencia en base de datos y notificación por mail a usuario y administración
  • Panel de administración con CRUD
  • Interfaz clara y responsiva
  • Diseño adaptable a dispositivos móviles
  • Persistencia y autenticación segura
  • Sistema de carrito para libros deseados por usuarios

Galería de imágenes

Tecnologías utilizadas

  • Frontend: HTML, CSS, JavaScript
  • Backend: PHP
  • Base de datos: MySQL
  • Herramientas: Git, Visual Studio Code, Composer, PHP Data Objects (PDO)
